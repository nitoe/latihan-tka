📝 Fitur Aplikasi Pengolah Kata

Aplikasi pengolah kata (seperti Microsoft Word, Google Docs) digunakan untuk membuat dokumen teks. Fitur-fitur utama:

FiturFungsiIkon
Bold (Tebal)Menebalkan teks agar menonjol (misal: judul)B
Italic (Miring)Memiringkan teks (misal: istilah asing, penekanan)I
Underline (Garis Bawah)Memberi garis bawah pada teksU
Align CenterMeratakan teks ke tengah halaman≡
Font SizeMengubah ukuran huruf12pt
Font ColorMengubah warna hurufA🎨
💡 Bold, Italic, Underline = Format TeksKetiganya digunakan khusus untuk memformat tampilan teks. Crop, Resize, Rotate bukan fitur pengolah kata — itu fitur pengolah gambar.

💾 Penyimpanan Data: Online vs Offline

☁️ Online (Cloud Storage)

Disimpan di server internet, bisa diakses dari mana saja.

  • Google Drive
  • OneDrive, Dropbox, iCloud
  • ✅ Bisa kerja di sekolah DAN di rumah tanpa flashdisk
  • ✅ Tidak hilang jika komputer rusak
  • ⚠️ Butuh koneksi internet

💽 Offline (Local Storage)

Disimpan di perangkat fisik, hanya bisa diakses dari perangkat itu.

  • Flashdisk — portable, kecil
  • Hard Disk Internal — di dalam komputer
  • DVD/CD — kapasitas terbatas
  • ✅ Tidak butuh internet
  • ⚠️ Bisa hilang/rusak
📝 Soal Klasik"Siswa ingin mengerjakan tugas di sekolah dan melanjutkan di rumah TANPA flashdisk" → Jawaban: Cloud Storage (Google Drive) karena bisa diakses dari mana saja via internet.

📂 Manajemen File dan Folder

Cara paling efektif mengorganisir banyak file yang tercampur:

  • ✅ Membuat sub-folder berdasarkan jenis file — misal: Folder "Gambar", "Dokumen", "Data"
  • ✅ Memberi nama file yang deskriptif — misal: "Tugas_IPAS_Bab3.docx"
  • ❌ Menghapus semua file lama (bisa kehilangan data penting)
  • ❌ Menyimpan semua di desktop (berantakan)
  • ❌ Mengubah nama semua jadi angka (sulit ditemukan)

🔧 Hardware vs Software

🔩 Hardware (Perangkat Keras)

Bisa disentuh secara fisik.

  • Mouse, Keyboard, Monitor
  • Hard Disk, RAM, CPU
  • Printer, Speaker, Webcam

💿 Software (Perangkat Lunak)

Tidak bisa disentuh — program/aplikasi.

  • Windows (Sistem Operasi)
  • Microsoft Word (Pengolah Kata)
  • Google Chrome (Browser)
⚠️ Jangan Tertukar!Windows dan Microsoft Word = software (tidak bisa disentuh). Mouse dan Hard Disk = hardware (bisa disentuh). Soal sering mencampur keduanya dalam satu pilihan.

🔍 Manfaat Internet untuk Pembelajaran

Internet sangat bermanfaat untuk belajar — asalkan digunakan dengan cara yang tepat.

  • ✅ Mencari di mesin pencari edukasi — cara paling efektif untuk mencari materi pelajaran
  • ❌ Bermain game online (bukan untuk belajar)
  • ❌ Mengobrol di media sosial (bukan tujuan akademik)

Sumber Informasi yang Terpercaya

✅ Terpercaya

  • Situs berita resmi (kompas.com, detik.com)
  • Website pemerintah (.go.id)
  • Situs lembaga pendidikan (.ac.id, .edu)

❌ Kurang Terpercaya

  • Blog pribadi tidak jelas sumbernya
  • Wiki/forum yang bisa diedit siapa saja
  • Pesan berantai WhatsApp tanpa sumber

Memverifikasi Hoaks

Saat menerima informasi mencurigakan (giveaway tanpa sumber, berita sensasional): cek di situs berita resmi sebelum menyebarkan. Jangan langsung share, jangan langsung klaim.

🔒 Keamanan Akun dan Password

Ciri-ciri Password yang Kuat

PasswordKuat / LemahAlasan

Jejak Digital

Jejak digital = semua rekam jejak aktivitas online (postingan, foto, komentar) yang sulit dihapus dan bisa ditemukan oleh siapa saja di masa depan. Foto tidak sopan yang diunggah sekarang → bisa merugikan reputasi saat mendaftar sekolah/kerja nanti.

🎣 Phishing dan Penipuan Daring

Phishing = upaya penipuan online yang menyamar sebagai pihak terpercaya untuk mencuri data pribadi.

Ciri-ciri Pesan Phishing

  • 🔗 Link website aneh — URL tidak resmi, banyak angka/huruf acak
  • 📋 Meminta data pribadi — password, nomor rekening, PIN
  • ⏰ Bahasa mendesak/urgent — "Segera klik!", "Akun Anda akan diblokir!"
  • ❌ Pengirim TIDAK jelas — bukan dari institusi resmi

Keamanan WiFi

⚠️ Perlu Waspada Ekstra

  • WiFi Publik di Kafe
  • WiFi Gratis Tanpa Password
  • Data bisa disadap oleh orang jahat

✅ Relatif Aman

  • WiFi Rumah (dilindungi password)
  • WiFi Sekolah (dikelola admin)

📱 Manfaat dan Risiko TIK

✅ Manfaat

  • Video Call — menjaga hubungan keluarga jarak jauh (kakek-nenek di luar pulau)
  • Akses informasi & belajar online
  • Komunikasi cepat via chat/email
  • Grup komunitas online untuk berbagi

❌ Risiko

  • Kecanduan gawai — bermain game hingga larut malam
  • Gangguan kesehatan fisik: mata lelah, kurang gerak
  • Gangguan kesehatan mental: stres karena medsos
  • Kurangnya tatap muka langsung
  • Kesalahan sistem pada teknologi baru (self-driving car)

⏰ Screen Time — Waktu Layar yang Sehat

Alokasi waktu penggunaan gawai yang paling sehat: saat waktu luang yang terbatas — bukan 24 jam, bukan saat jam belajar, dan BUKAN saat jam tidur.

  • ✅ Batasi waktu layar — gunakan hanya saat waktu luang
  • ✅ Jeda setiap 30-60 menit untuk istirahat mata
  • ❌ Bermain game sampai larut malam → gangguan tidur dan mata
  • ❌ Menggunakan gawai saat jam belajar → konsentrasi terpecah

🛑 Cyberbullying dan Etika Digital

Cyberbullying

Cyberbullying = perundungan yang terjadi di dunia digital (media sosial, grup chat, komentar). Jika melihat teman diejek secara kasar online, sikap yang paling tepat: melaporkan dan menegur — bukan ikut mengejek, diam saja, atau menyebarkan.

Etika Berkomunikasi di Grup Chat

PerilakuEtis / Tidak Etis
Mengucapkan terima kasih✅ Etis
Mengirim pesan relevan dan sopan✅ Etis
Mengirim spam / pesan berulang❌ Tidak Etis
Mengejek/menghina teman❌ Tidak Etis
Menyebar hoaks tanpa verifikasi❌ Tidak Etis

©️ Hak Cipta dan Penggunaan Karya Digital

Saat menggunakan gambar/karya dari internet untuk tugas:

  • ✅ Menyebutkan sumber — tulis dari mana gambar tersebut diambil
  • ✅ Meminta izin pemilik — jika karya tersebut memiliki hak cipta
  • ❌ Copy-paste tanpa kredit (mengabaikan hak cipta)
  • ❌ Mengaku sebagai karya sendiri (plagiarisme)

🌐 Dampak Teknologi terhadap Interaksi Sosial

Teknologi mengubah cara manusia berinteraksi — ada sisi positif dan negatif:

  • ✅ Bertemu virtual — bisa berkomunikasi jarak jauh
  • ✅ Grup komunitas online — berbagi minat/hobi
  • ⚠️ Kurang tatap muka langsung — interaksi langsung berkurang
  • ⚠️ Ketergantungan pada gawai — sulit lepas dari layar

📝 Ringkasan Poin Kunci untuk Ujian

Aplikasi & File

  • Format teks: Bold, Italic, Underline. Crop/Resize = pengolah gambar, bukan pengolah kata
  • Ekstensi: .docx = dokumen, .xlsx = spreadsheet, .jpg = gambar, .mp3/.mp4 = multimedia
  • Cloud Storage (Google Drive) → akses dari mana saja tanpa flashdisk
  • Organisir file → buat sub-folder berdasarkan jenis
  • SUM/AutoSum → menghitung total di spreadsheet

Sistem Komputer

  • Alur: Input → Proses (CPU) → Output
  • Input: keyboard, mouse, scanner, microphone, webcam
  • Output: monitor, printer, speaker
  • CPU = "otak" komputer (pemroses utama)
  • Hardware = bisa disentuh. Software = program/aplikasi
  • RAM & Hard Disk = internal. Monitor & Keyboard = eksternal

Jaringan & Internet

  • Password kuat: huruf besar + kecil + angka + simbol (Sekolah2026!)
  • Data sensitif: alamat rumah, password → JANGAN dibagikan online
  • Jejak digital bersifat permanen → bisa merugikan di masa depan
  • Ciri phishing: link aneh + minta data pribadi + bahasa mendesak
  • WiFi publik/gratis = perlu waspada ekstra
  • Sumber terpercaya: situs berita resmi, .go.id

Dampak Sosial

  • Video Call = manfaat TIK untuk keluarga jarak jauh
  • Screen time sehat = saat waktu luang terbatas, bukan 24 jam
  • Cyberbullying → laporkan dan tegur
  • Hak cipta: sebut sumber + minta izin
  • Etis: ucap terima kasih, sopan. Tidak etis: spam, ejek, hoaks
  • Warga digital bertanggung jawab: verifikasi berita, bijak waktu layar
